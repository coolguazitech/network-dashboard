## 做了什麼

NETORA 是歲修驗收自動化平台，自動採集網路設備資料、判定健康狀態、產出驗收報告。

**核心能力**：
- **8 大指標全自動檢查**：光模組、風扇、電源、版本、Uplink、Port-Channel、CRC Error、Ping — 以前缺的都補齊了
- **協作查案系統**：每台 client 自動建案、Ping 燈號即時顯示、自動結案/重開、PM 可指派分工
- **拓樸視覺化**：自動繪製設備連線圖，期望 vs 實際一目了然
- **RBAC 權限管理**：ROOT / PM / GUEST 三級角色，權責分明
- **一鍵報告匯出**：HTML 驗收報告直接下載，不用手動彙整

## 解決什麼問題

| 以前 | 現在 |
|------|------|
| 進 OTJH → bastion → docker 指令操作 | 瀏覽器直接開，上 TKS 後更方便 |
| 缺光模組、CRC、版本驗收，靠人工逐筆比對 | 8 大指標自動採集 + 自動判定 |
| 沒有權限控管、沒有查案流程 | 三級角色 + 案件追蹤 + 系統日誌 |
| 驗收報告手工彙整 | 一鍵產出 |

## 近期技術改進（v2.19.4 ~ v2.19.15）

1. **SNMP 引擎重寫**：用 subprocess 取代 pysnmp，50 台以上併發不再崩潰，200 台穩定運行
2. **採集效能提升**：每台設備從 200 秒降到 12 秒，200 台全部跑完只要 2 分鐘
3. **拓樸精確比對**：從「兩台設備有連線就算通過」改為「逐條 interface 精確比對」，不匹配的 interface 正確標紅
4. **部署穩定性**：所有 DB migration 加入防禦性檢查，任何部署路徑都不會報錯
